Hi again. This is me practicing git commands to further master them.

<b>git clone</b> - Downloads a project with its entire history from a remote repository.

<b>git add</b> - Stages a file for the next commit. Use git add . to stage all modified and new files in the current directory and its subdirectories.

<b>git status</b> - Commands related to checking status or log history

<b>git commit -m</b> - Records the staged changes as a new commit with a descriptive message.

<b>git push</b> - Uploads local branch commits to the remote repository.
